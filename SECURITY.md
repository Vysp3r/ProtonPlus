# Security Policy

## Supported Versions

Currently, only the latest release of ProtonPlus is supported for security updates.

| Version | Supported |
| ------- |-----------|
| 0.5.x   | ✅         |
| < 0.5.x | ❌       |

## Reporting a Vulnerability

If you've found a security vulnerability in ProtonPlus, please let us know!
You can report it by:

1. **Emailing us**: Send an email to [dev@vysp3r.com](mailto:dev@vysp3r.com).
2. **Private Disclosure**: If you are using GitHub, you can use the [Private Vulnerability Reporting](https://docs.github.com/en/code-security/security-advisories/guidelines-for-reporting-a-security-vulnerability#reporting-a-vulnerability-to-a-maintainer) feature.

Please **do not** open a public issue for security vulnerabilities.

We'll do our best to respond quickly and keep you updated on our progress. Thank you for helping keep ProtonPlus secure!
